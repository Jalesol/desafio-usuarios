# Listagem de Usuários

Projeto desenvolvido como parte de teste técnico. A aplicação permite listar, buscar, cadastrar e editar usuários, com persistência de dados no navegador.

## 🚀 Tecnologias utilizadas

- Angular
- TypeScript
- SCSS
- LocalStorage

## 📋 Funcionalidades

- Listagem de usuários em cards
- Exibição de nome, e-mail, telefone e CPF
- Busca por nome com debounce de 300ms
- Cadastro de novos usuários
- Edição de usuários
- Persistência dos dados após recarregar a página
- Loading inicial ao carregar os usuários
